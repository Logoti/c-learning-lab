# C Learning Lab

Repositório dedicado ao estudo e prática de programação em C.

Aqui organizo exercícios e problemas por temas, com foco em:
- lógica de programação
- estruturas de controle
- arrays e funções
- ponteiros e manipulação de memória

## Estrutura
- Basics: entrada, saída e condicionais
- Loops: estruturas de repetição
- Arrays: manipulação de vetores
- Functions: modularização de código
- Pointers: conceitos de memória

## Objetivo
Construir base sólida em programação e resolução de problemas.
